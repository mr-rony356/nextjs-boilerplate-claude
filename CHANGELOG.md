# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-14

### Added

- Initial release of Next.js 15 Gold Master Boilerplate
- Next.js 15 with App Router, Server Actions, and PPR
- Tailwind CSS v4 with CSS-first configuration
- Shadcn/UI component library integration
- Drizzle ORM with Neon Postgres support
- Auth.js v5 with multiple OAuth providers (GitHub, Google, Credentials)
- TypeScript with strict mode enabled
- Complete authentication flow (signin, signup, signout)
- Protected dashboard with navigation
- Profile and settings pages
- Database schema with Drizzle-Zod integration
- Server Actions for data mutations
- Middleware for route protection
- Docker and Docker Compose configuration
- GitHub Actions CI/CD workflows
- Comprehensive documentation (README, SETUP, CONTRIBUTING)
- ESLint and Prettier configuration
- Issue and PR templates

### Developer Experience

- Hot reload with Turbopack
- Type-safe database queries
- Automatic TypeScript type generation from schemas
- Database migrations with Drizzle Kit
- Drizzle Studio for visual database management
- Form validation with Zod
- Toast notifications with Sonner

### UI/UX

- Responsive design
- Dark mode support (via Tailwind)
- Accessible components
- Beautiful landing page
- Clean dashboard interface
